如何下载配置JDK

2024-11-28
阅读时长: 2 分钟

安装 JDK（Java Development Kit）并配置环境变量是学习 Java 编程的第一步。以下是详细步骤，适用于 Windows 和 Mac/Linux 系统：

Windows 系统

1. 下载 JDK

  1. 前往 Oracle 官方网站或 OpenJDK 官网：
    • Oracle JDK：https://www.oracle.com/java/technologies/javase-downloads.html
    • OpenJDK：https://openjdk.org/
  2. 下载适合你的系统（Windows x64）的 JDK 安装包。

2. 安装 JDK

  1. 双击下载的安装包，启动安装程序。
  2. 按照提示进行安装：
    • 选择安装路径（例如 C:\Program Files\Java\jdk-XX.X.X）。
    • 安装完成后记住安装路径。

3. 配置环境变量

  1. 打开环境变量设置：
    • 右键“此电脑”或“我的电脑”，选择“属性”。
    • 点击“高级系统设置”。
    • 在“系统属性”窗口中，选择“高级”选项卡，点击“环境变量”。
  2. 设置 JAVA_HOME：
    • 点击“系统变量”下的“新建”按钮。
    • 变量名：JAVA_HOME
    • 变量值：输入 JDK 的安装路径，例如 C:\Program Files\Java\jdk-XX.X.X
  3. 配置 PATH 变量：
    • 在“系统变量”中找到 Path，点击“编辑”。
    • 新增一行，输入 %JAVA_HOME%\bin
  4. 配置 CLASSPATH（可选）：
    • 点击“新建”变量。
    • 变量名：CLASSPATH
    • 变量值：.:%JAVA_HOME%\lib\tools.jar;%JAVA_HOME%\lib\dt.jar （注意：路径分隔符 ;）

4. 验证安装

  1. 打开命令提示符（Win + R，输入 cmd）。
  2. 输入以下命令：
    • 检查 Java 版本：java -version
    • 检查编译器版本：javac -version

如果显示 JDK 版本信息，说明安装和配置成功。

Mac/Linux 系统

1. 使用包管理器安装 JDK（推荐）

Mac：

  1. 打开终端，运行以下命令：

    1

    brew install openjdk

  2. 查看安装路径：

    1

    /usr/libexec/java_home

Linux：

  1. 使用包管理器安装：

    • Ubuntu/Debian：

      1
2

      sudo apt update
sudo apt install openjdk-XX-jdk

    • CentOS/RHEL：

      1

      sudo yum install java-XX-openjdk-devel

  2. 验证安装：

    1

    java -version

2. 手动下载与安装

  1. 前往 Oracle 或 OpenJDK 网站，下载对应的 JDK 安装包（如 .dmg、.tar.gz）。

  2. 解压（Linux）：

    1
2

    tar -xvf jdk-XX.X.X.tar.gz
sudo mv jdk-XX.X.X /usr/lib/jvm

3. 配置环境变量

Mac：

  1. 打开 ~/.zshrc 或 ~/.bash_profile 文件。

  2. 添加以下内容：

    1
2

    export JAVA_HOME=$(/usr/libexec/java_home)
export PATH=$JAVA_HOME/bin:$PATH

  3. 保存并刷新终端配置：

    1

    source ~/.zshrc

Linux：

  1. 编辑

    1

    ~/.bashrc

    或

    1

    ~/.bash_profile

    文件：

    1
2

    export JAVA_HOME=/usr/lib/jvm/jdk-XX.X.X
export PATH=$JAVA_HOME/bin:$PATH

  2. 保存并生效：

    1

    source ~/.bashrc

4. 验证安装

  1. 运行以下命令：

    1
2

    java -version
javac -version

  2. 如果能正确显示版本信息，说明安装成功。
