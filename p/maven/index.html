如何安装配置Maven

2024-11-29
阅读时长: 2 分钟

Windows 系统

1. 下载 Maven

  1. 前往 Apache Maven 官方下载页面： https://maven.apache.org/download.cgi
  2. 下载二进制 ZIP 包（Binary zip archive），例如：apache-maven-3.x.x-bin.zip。

2. 解压 Maven

  1. 将下载的 ZIP 文件解压到一个目录，例如：C:\Program Files\Maven。
  2. 解压完成后，记住 Maven 的安装路径（如 C:\Program Files\Maven\apache-maven-3.x.x）。

3. 配置环境变量

  1. 打开环境变量设置：
    • 右键“此电脑”或“我的电脑”，选择“属性”。
    • 点击“高级系统设置”，然后点击“环境变量”。
  2. 设置 MAVEN_HOME 变量：
    • 点击“系统变量”中的“新建”。
    • 变量名：MAVEN_HOME
    • 变量值：Maven 的安装路径，例如 C:\Program Files\Maven\apache-maven-3.x.x
  3. 编辑 Path 变量：
    • 找到“系统变量”中的 Path，点击“编辑”。
    • 新增一行，输入 %MAVEN_HOME%\bin。
  4. 保存并退出。

  1. 4. 验证安装

    1. 打开命令提示符（Win + R，输入 cmd）。

    2. 输入以下命令：

      1

      mvn -v

    3. 如果输出 Maven 版本和 Java 信息，则安装成功。

      1
2
3

      Apache Maven 3.x.x
Java version: 1.x.x
Java home: C:\Program Files\Java\jdk-xx.x

Mac/Linux 系统

1. 使用包管理器安装（推荐）

Mac：

  1. 打开终端，运行以下命令安装 Maven：

    1

    brew install maven

Linux：

  1. 对于基于 Debian 的系统（如 Ubuntu）：

    1
2

    sudo apt update
sudo apt install maven

  2. 对于基于 Red Hat 的系统（如 CentOS）：

    1

    sudo yum install maven

  3. 验证安装：

    1

    mvn -v

2. 手动下载并安装

  1. 下载 Maven： 前往 Apache Maven 官方网站，下载二进制文件。

  2. 解压文件：

    1
2

    tar -xvf apache-maven-3.x.x-bin.tar.gz
sudo mv apache-maven-3.x.x /opt/maven

  3. 配置环境变量：

    • 打开 ~/.bashrc（或 ~/.zshrc），添加以下内容：

      1
2

      export MAVEN_HOME=/opt/maven/apache-maven-3.x.x
export PATH=$MAVEN_HOME/bin:$PATH

  4. 保存并生效：

    1

    source ~/.bashrc

  5. 验证安装：

    1

    mvn -v

配置 Maven

Maven 的默认配置文件位于 conf/settings.xml。以下是一些常见的配置选项：

1. 设置本地仓库

默认情况下，Maven 的本地仓库位于 ~/.m2/repository。可以在 settings.xml 中更改：

1
2
3

<settings>
  <localRepository>C:/maven/repo</localRepository>
</settings>

2. 配置镜像

为提高依赖下载速度，可以配置国内镜像（例如阿里云）：

1
2
3
4
5
6
7
8

<mirrors>
  <mirror>
    <id>aliyun-maven</id>
    <mirrorOf>*</mirrorOf>
    <name>aliyun maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>

3. 配置 JDK 版本

在 pom.xml 中指定 Java 版本：

1
2
3
4

<properties>
  <maven.compiler.source>1.8</maven.compiler.source>
  <maven.compiler.target>1.8</maven.compiler.target>
</properties>
