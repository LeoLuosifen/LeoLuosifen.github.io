如何安装配置MySQL

2024-11-29
阅读时长: 3 分钟

Windows 系统

1. 下载 MySQL

  1. 前往 MySQL 官方下载页面：https://dev.mysql.com/downloads/
  2. 选择 MySQL Community Server。
  3. 下载适合你系统的安装程序（一般是 MSI Installer）。

2. 安装 MySQL

  1. 启动安装程序：
    • 双击 MSI 文件，启动安装向导。
  2. 选择安装类型：
    • 一般选择 Custom，以便自定义安装路径。
    • 默认安装路径通常是：C:\Program Files\MySQL\MySQL Server X.X。
  3. 配置服务：
    • 设置 MySQL 服务的端口（默认是 3306）。
    • 选择默认用户身份验证插件（推荐：caching_sha2_password）。
    • 设置 root 用户的密码，并根据需要创建其他用户。
  4. 完成安装：
    • 安装完成后，记住配置的路径和 root 密码。

3. 配置环境变量

为了方便使用 MySQL 的命令行工具（如 mysql 和 mysqldump），需要配置环境变量。

  1. 打开环境变量设置：
    • 右键“此电脑”或“我的电脑”，选择“属性”。
    • 点击“高级系统设置” -> “环境变量”。
  2. 编辑 Path 变量：
    • 在系统变量中找到 Path，点击“编辑”。
    • 新增 MySQL 安装目录的 bin 路径，例如：C:\Program Files\MySQL\MySQL Server X.X\bin。
  3. 保存并退出。

4. 验证安装

  1. 打开命令提示符（Win + R，输入 cmd）。

  2. 输入以下命令检查版本：

    1

    mysql --version

  3. 登录 MySQL：

    1

    mysql -u root -p

    输入安装时设置的 root 密码，成功登录后将看到 MySQL 提示符 (mysql> )。

Mac/Linux 系统

1. 使用包管理器安装

Mac：

  1. 打开终端，运行以下命令：

    1

    brew install mysql

Linux：

  1. 对于基于 Debian 的系统（如 Ubuntu）：

    1
2

    sudo apt update
sudo apt install mysql-server

  2. 对于基于 Red Hat 的系统（如 CentOS）：

    1

    sudo yum install mysql-server

2. 启动 MySQL 服务

Mac/Linux：

  1. 启动 MySQL 服务：

    1

    sudo service mysql start

  2. 检查 MySQL 服务状态：

    1

    sudo service mysql status

3. 安全配置

  1. 运行 MySQL 安全脚本：

    1

    sudo mysql_secure_installation

  2. 按提示进行以下配置：

    • 设置 root 用户密码。
    • 移除匿名用户。
    • 禁止 root 用户远程登录。
    • 移除测试数据库。

4. 登录 MySQL

  1. 登录 MySQL：

    1

    mysql -u root -p

  2. 输入 root 密码后即可进入 MySQL 命令行。

图形化管理工具

为了更方便地管理 MySQL，可以使用以下工具：

  1. MySQL Workbench：
    • 官方提供的图形化管理工具。
    • 可从 MySQL Workbench 下载。
  2. 第三方工具：
    • phpMyAdmin（基于 Web 的管理工具）。
    • DBeaver（跨平台数据库管理工具）。
    • Navicat

常用配置

1. 修改 MySQL 配置文件

MySQL 配置文件的路径可能是：

  • Windows：C:\ProgramData\MySQL\MySQL Server X.X\my.ini
  • Linux/Mac：/etc/mysql/my.cnf 或 /etc/my.cnf

Linux/Mac： 

1

常见问题

  1. MySQL 无法启动或连接失败

    • 检查 MySQL 服务是否启动：

      1

      sudo service mysql status

    • 检查防火墙是否阻止了 3306 端口。

  2. 忘记 root 密码

    • 参考以下步骤重置密码：

      1. 停止 MySQL 服务：

        1

        sudo service mysql stop

      2. 以安全模式启动：

        1

        sudo mysqld_safe --skip-grant-tables &

      3. 登录 MySQL 并重置密码：

        1
2
3

        mysql -u root
ALTER USER 'root'@'localhost' IDENTIFIED BY 'newpassowrd';
FLUSH PRIVILEGES;

  3. 字符集问题

    • 确保 MySQL 使用 utf8mb4 以支持完整的 Unicode 字符。
